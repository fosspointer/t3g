# T3G: Customizable tic-tac-toe implementation in C++

This is a console program which offers a classic 'tic-tac-toe' game experience, but with extensibility to arbitrary proportions (literally speaking).

## Requirements

- Interest in tic-tac-toe and/or in-console gamse
- CMake 3.16
- A C++ 23 compiler

## Configuration

Currently, no global configuration method exists. However, options exist to change the board dimensions, alter the required same-player-pile sequence for winnning, as well as changing the symbol for the two players. For more info, run this game with the `--help` flag (every option is documented there)
